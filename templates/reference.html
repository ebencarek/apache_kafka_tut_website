Reference


Topic Category or feed to which records are published, made up of 1 or more partitions
Partition Ordered, immutable sequence of records that is append-only
Record A piece of data stored in a partition, consists of a key, value, and timestamp
Replication factor The number of replicas kept for each partition in a topic
Leader The node in the Kafka cluster that handles all read/write operations to a particular partition
Follower Any node that maintains a copy of a particular partition, may become leader if current leader fails
Consumer System that subscribes to one or more topics to process the streams of records published to them
Consumer group Logical grouping of consumer processes, where each record in a partition is only sent to one consumer per consumer group
Producer System that publishes records to one or more topics
Cluster Collection of Kafka nodes
Broker A node in the Kafka cluster
ZooKeeper Service used by Kafka to maintain state within the cluster
